# Neuro-Symbolic Logic Validator
- A simple web application for neuro-symbolic logic validator using Groq API.
# The Link to the Demo Web Application
- https://neuro-symbolic-logic-validator.streamlit.app
# Requirement
- Python 3.10
# How to Use
1. Download and extract the files.
2. Run the command prompt as an administrator.
3. Install the required packages using the following command:
   
   ```Bash
   pip install -r requirements.txt
   
4. In the command prompt, navigate to the folder in which the file `neuro_symbolic_logic_validator.py` is located. For example:
   
   ```Bash
   cd /users/user_name/Downloads/Neuro-Symbolic-Logic-Validator-master/

6. Run `neuro_symbolic_logic_validator.py` using the following command:
    
   ```Bash
   streamlit run neuro_symbolic_logic_validator.py

7. Open your web browser and go to `http://localhost:8501` to access the web application.
